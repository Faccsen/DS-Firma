# DS-Firma 🛒

> Vollautomatisches Dropshipping — von TikTok-Trend bis Shopify-Listing

## Was ist die DS-Firma?

Eine KI-betriebene Dropshipping-Firma die täglich TikTok scannt,
profitable Produkte identifiziert und automatisch in einem Shopify-Shop listet.

## Org Chart

```
CEO (Claude Opus 4.6)
├── Trend Scout (Claude Sonnet 4.6)        — Täglicher TikTok-Scan
├── Product Analyst (Claude Sonnet 4.6)   — Margen & Lieferanten-Analyse
├── Store Manager (Claude Sonnet 4.6)     — Shopify-Management
│   └── Ad Copywriter (Claude Haiku 4.5)  — Produkttexte & Ads
└── Customer Service (Claude Haiku 4.5)   — Kundenbetreuung
```

## Wie es funktioniert

```
1. Trend Scout scannt täglich TikTok (#tiktokmademebuyit etc.)
           ↓
2. Product Analyst bewertet Marge & Lieferanten
           ↓
3. CEO entscheidet: Listen oder Skip?
           ↓
4. Store Manager & Ad Copywriter erstellen das Shopify-Listing
           ↓
5. Customer Service betreut Kunden & liefert Feedback
```

## Ziel

100.000€ Monatsumsatz durch vollautomatisches Dropshipping auf Shopify.
